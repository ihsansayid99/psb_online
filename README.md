# PSB ONLINE

PSB ONLINE adalah Sistem informasi penerimaan siswa baru. yang dibuat memakai PHP Native dan dependecies lainnya.

## Installation

git clone project psb_online ini.

```bash
composer install
```

## Folder
- app
- config
- images
- assets
- helpers
- pdf_siswa
